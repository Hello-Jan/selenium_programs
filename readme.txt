一：项目概况
本项目一共做了两个自动化测试：测试智联招聘职位搜索框的功能；测试我要自学网的登录功能。
使用到的软件：python 3.8.0
使用到的库：selenium==2.48.0;xlwt==1.3.0;HTMLTestRunner为网络上下载并修改的版本
使用的浏览器：Firefox 35.0.1


二项目介绍：
测试智联招聘的Job类（job.py）中有test方法、print方法、export_excel方法。其中print方法可以将搜索到的职位（部分）信息打印出来，export_excel方法可以将结果输出到excel表格(job.xls)。单元测试(test_job)中主要用到test方法。
测试我要自学网的测试功能：通过输入不同情况的账号密码验证登录页面的功能是否满足需求。
